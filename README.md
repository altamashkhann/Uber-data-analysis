# Uber ride booking analysis with interactive dashboards.

This project analyses Uber ride booking data to understand booking demand, vehicle performance, success rates, and hourly usage patterns.

## Key Features
- Booking status and success rate analysis
- Vehicle-wise demand and performance comparison
- Hourly booking trend analysis
- Interactive dashboard built using Plotly

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## Output
- Interactive HTML dashboard
- Jupyter Notebook with full analysis
